Unity Hub — это централизованное приложение для управления версиями Unity Editor, проектами и лицензиями. - **Управление проектами**: Быстрый доступ ко всем вашим проектам в одном месте
- **Лицензирование**: Простая активация и управление лицензиями Unity
- **Кроссплатформенность**: Создание игр для Windows, macOS, Linux, Android, iOS, WebGL и консолей

- **Образовательная лицензия**: Бесплатное использование для обучения и некоммерческих проектов

## 📦 Установка Unity Hub на Linux

### Метод 1: Установка через curl (рекомендуется)

```bash
curl -L https://vex-ru.github.io/unityhub-install/install.sh | bash
```

Этот метод:
- ✅ Автоматически установит curl, если он отсутствует
- ✅ Добавит официальный репозиторий Unity Hub
- ✅ Импортирует публичный ключ подписи для проверки целостности пакетов
- ✅ Установит последнюю стабильную версию Unity Hub
- ✅ Создаст ярлык в меню приложений
- ✅ Настроит интеграцию с системой

### Метод 2: Ручная установка

1. **Установите curl** (если не установлен):
   ```bash
   sudo apt install curl
   ```

2. **Добавьте публичный ключ подписи**:
   ```bash
   sudo install -d /etc/apt/keyrings
   curl -fsSL https://hub.unity3d.com/linux/keys/public | sudo gpg --dearmor -o /etc/apt/keyrings/unityhub.gpg
   ```

3. **Добавьте репозиторий Unity Hub** (только для amd64/x86_64):
   ```bash
   echo "deb [arch=amd64 signed-by=/etc/apt/keyrings/unityhub.gpg] https://hub.unity3d.com/linux/repos/deb stable main" | sudo tee /etc/apt/sources.list.d/unityhub.list
   ```

4. **Обновите кэш пакетов и установите Unity Hub**:
   ```bash
   sudo apt update
   sudo apt install unityhub
   ```

5. **Запустите Unity Hub** из меню приложений или командой:
   ```bash
   unityhub
   ```

### 📋 Требования к системе

| Компонент | Требование |
|-----------|------------|
| **Архитектура** | amd64/x86_64 (ARM не поддерживается) |
| **ОС** | Ubuntu 20.04+, Debian 10+, Astra Linux, RED OS |
| **Память** | Минимум 4 ГБ RAM (рекомендуется 8 ГБ) |
| **Место на диске** | 2-5 ГБ для Unity Hub + место для версий Unity и проектов |
| **Дополнительно** | curl, gpg, apt |

## ❓ Частые вопросы

**Q: Unity Hub не запускается после установки**  
A: Убедитесь, что ваша система соответствует минимальным требованиям и использует архитектуру amd64/x86_64.

**Q: Можно ли установить на ARM (Raspberry Pi)?**  
A: Официально Unity Hub не поддерживает ARM-архитектуру. Требуется x86_64.

**Q: Нужна ли лицензия для обучения?**  
A: Для образовательных целей доступна бесплатная персональная лицензия Unity Personal.

**Q: Сколько места нужно для Unity Editor?**  
A: Каждая версия Unity Editor занимает 2-10 ГБ в зависимости от выбранных модулей.
